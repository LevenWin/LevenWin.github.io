1和0和计算机

因为人类能够掌控事物的正反面，所以计算机诞生了

最近看了《编码-隐匿在计算机软硬件背后的语言》这本书，让我对计算机的构成和运行了有了更深入的了解。计算机是纯逻辑的工具，暂时不带任何意识和情感，所有的运作都是基于1和0。当人们掌握一件物件的正反面的时候，计算机就诞生了。

继电器与门

对于计算机，输入为1或者0，中间经过转换后输出1或者0，中间转换的逻辑就是依赖有规律的或与门的处理然后输出。例如我们希望输入1，电脑输出0，.我们可以利用比较原始的继电器实现这点。假设开关放下代表1，开关打开代表0， 灯亮代表1，灯灭代表0。实际上我们实现了反向器

关 -> 灯灭 ： 1 -> 0
开 -> 灯亮 ： 0 -> 1

自动化

当在RAM中存入指令和数据，CPU根据读到的指令和数据在时钟信号的驱动下做数据运算，存储和输出。我们平时用高级语言写的代码，在编译成机器语言之后，就是CPU能识别的指令和数据了。

总线

CPU与周边设备的信息交流就是通过总线来运作的，一般为了效率会在CPU等设备内部也会存在一些微总线。设备之间可以通过中断来决定总线的使用分配。

操作系统

为了人类更有效的控制计算机，人们发明了操作系统。从最开始的纸片打孔来控制计算机的运行，到现在分时复用，多任务，多线程来更有效的分配计算机资源。为了解决内存不够，碎片化等问题，操作系统又引入了虚拟内存技术。另外操作系统提供了高级语言的编辑，编译功能，让人们可以做出更自定义化的计算机操控程序。更伟大的是计算机中图形界面的使用，让计算机走进了千家万户。

计算机是人类第三次工业革命的主力，给我们的未来带来了无限的想象空间。但是计算机的思想早在几百年前就产生过萌芽，由于当时的工业的发展落后，甚至还处于启蒙状态，物理材料技术不够支持及时迅速的自动化操作。直到了电学和材料学的深入发展，才让计算机能够落实到具体的形态，以接近光电的速度运行。
